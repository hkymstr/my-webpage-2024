Bouncing Ball Game

A simple but challenging game where you control a cursor with arrow keys while avoiding a bouncing ball. The longer you survive, the higher your score!

How to Play:

  1. Download the game from GitHub
  2. Install Python if you haven't already (Download Python)
  3. Install the required package by running: pip install pygame
  4. Run the game using: python bouncing_ball.py

Controls:

  • Use arrow keys to move the blue cursor
  • Avoid touching the red bouncing ball
  • Press ESC to quit
  • Press R to restart after game over
